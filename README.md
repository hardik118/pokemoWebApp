# 🐉 Pokémon Next.js App  

This is a **Pokémon-themed Next.js application** that fetches and displays Pokémon data.  

## 📌 Features  

✅ **Fetch & Display Pokémon Data** from an API  
✅ **External Image Support** (Allows images from GitHub)  
✅ **Fast & Scalable Next.js Setup**
